Question-1: What are the different ways to select an element in the DOM?

Below are the method to select element in the DOM

  1. document.getElementById
  2. document.getElementsByClassName
  3. document.getElementsByTagName
  4. document.querySelector
  5. document.querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Difference between innerHTML, innerText and textContent is given below

  1. innerHTML: It gets all the element including markup inside the selected element.
  2. innerText: It gets only the text content which will appear on the screen.
  3. textContent: It gets all the text content which are available inside the selected element including the hidden element.

Question-3: What is event delegation in the DOM?

By using event delegation we can handle events triggered on the child elements from the same event listener used in the parent element.

Question-4: What is event bubbling in the DOM?

Event triggered in any child element can drive to the upper elements towards the dom tree and can respond to the event attached to any parent element is called Event Bubbling.
